## 🚀 Instalação
```
# Clone este repositório
$ git clone https://github.com/michelewtc/devquiz-flutter-nlw5

# Acesse a pasta do projeto
$ cd devquiz

# Instale as dependências
$ flutter pub get

# Rodar o aplicativo
$ flutter run
```
